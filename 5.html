2022 로드맵

자바스크립트, 타입스크립트, Node.js 또는 파이썬, 백엔드는 여전히 JSP가 강세(국내한정)

프런트엔드

  1. HTML5, CSS, JS (ES5, ES6...) 의 완벽한 이해
  2. 자바스크립트는 프로토타입, 클로저, 호이스팅 개념 완벽이해
  3. 타입스크립트는 이제 필수
  4. React, Vue, Angular2.0 3가지 중에 1개는 필수이나 React 추천 (Vue도 나쁘진않음)
  5. SPA -> SSR(서버사이드렌더링) Next.js(리액트), Nuxt.js(뷰) 사용법 숙지
  6. 또 Svelet이 떠오름, (권장사항)
  7. Git과 NPM은 (Yarn이 인기) 필수
  8. AWS, Azura와 더불어 Docker사용 등의 데브옵스가 중요해짐(프론트엔드건 백엔드건 필수)
  9. 부트스트랩은 기본, Tailwind CSS 같은 프레임워크는 권장사항
  10. SASS 같은 CSS프리픽스는 권장사항, POST CSS 떠오름 (필수는 아니나 현업을 위해)
  11. API사용법 익히기
  12. VSCODE 대동단결 그러나 회사가 여유있으면 젯브레인(웹스톰,인텔리제이) 좋으나 비쌈 나도 제대로 못써봄
  13. React Native, Flutter 같은 네이티브변환 프레임워크도 권장사항 (Flutter를 위해서는 Dart 언어 익혀야함),
  14. PWA가 떠오름 (웹페이지가 이제 앱을 대체?)

백엔드

  1. 백엔드도 자바스크립트,타입스크립트는 필수!
  2. Node.js 또는 파이썬은 필수, JSP는 국내한정 권장사항
  3. API 설계, 얼마나 API구조를 아름답게 짜는게 관건
  4. 데이터베이스 실력이 백엔드의 연봉을 좌지우지함
  5. GraphQL, NoSQL, SQL은 여전히 대세
  6. 웹어셈블리 (GO, RUST, C,C++)
  7. 파싱 -> 데이터 이쁘게 분류
  8. 머신러닝, 딥러닝, AI
  9. 여전히 중요한 데브옵스 (AWS, Azura, Docker, kubernetes 등등)
  10. 한번 배운 백엔드는 10년은 거뜬할지모르나 프론트엔드와 다르게 깊이가 다름

  2. 진정한 풀스택을 원한다면 10년은 걸림
